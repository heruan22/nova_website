# nova_website

A simple and elegant website to display git repository history.

## Features

- Clean and modern user interface
- Displays git commit history
- Responsive design
- Easy to use

## Usage

Open `index.html` in a web browser to view the git log history.

## Local Development

Simply open the `index.html` file in your preferred web browser. No build process or server required.

## Technologies Used

- HTML5
- CSS3
- Responsive design with modern CSS Grid and Flexbox
